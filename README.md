# Selenium Automation Framework

## Project Overview
This project is a Selenium Automation Framework developed using Java, Maven, TestNG, and Selenium WebDriver.

## Features
- Page Object Model (POM)
- TestNG Framework
- Extent Reports
- Screenshot Capture
- Excel Data-Driven Testing
- Test Listeners
- Maven Project Structure

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI
- Extent Reports
- Git & GitHub
